# 详细修复流程

## 1. 诊断设备

```bash
lsusb
nut-scanner -U
upsc 1100231@localhost
```

## 2. 修复 ups.conf

添加：
```
[1100231]
	driver = nutdrv_qx
	port = auto
	subdriver = krauler
	langid_fix = 0x0409
	vendorid = 0001
	productid = 0000
	desc = "Pudidun PD1000"
```
